class superbio.client.Client(email, password, token=None)[source]¶

Bases: object

Main client class for interacting with the Superbio API.

The client provides programmatic access to the Superbio platform, allowing users to: - List and browse available apps - Get app configurations and parameters - Submit and manage computational jobs - Track job status and retrieve results - Monitor balances

Note

You must first create an account at https://app.superbio.ai before using this client. Your email and password from the platform will be used to authenticate API requests.

Parameters:

  • email (str)

  • password (str)

  • token (str)

Submit a new job to the Superbio platform.

Parameters:

  • app_id (str) – ID of the application to run

  • running_mode (str) – Either “gpu” or “cpu”

  • config (dict, optional) – Job configuration parameters

  • local_files (dict, optional) – Mapping of file keys to local file paths

  • remote_file_source_data (dict, optional) –

    Remote file source configuration Example for S3: {

    ”file_key”: [{

    “protocol”: “s3”, “credentials”: {

    ”aws_access_key_id”: “YOUR_ACCESS_KEY”, “aws_secret_access_key”: “YOUR_SECRET_KEY”

    }, “path”: “bucket/path/to/file.csv”

    }]

    }

  • datahub_file_data (dict, optional) –

    Mapping of file keys to lists of datahub file paths Example: {

    ”file_key”: [“path/to/datahub/file.csv”]

    }

  • validate (bool, optional) – Whether to validate inputs before submission

Returns:

Response from the API containing job details

Example: {

”job_id”: “job_123abc…”

}

Return type:

dict

Raises:

Exception – If validation fails or job submission fails
